Bubble Sort

  • Graphic for Functionality

  • Implementation in Python
    def bubble_sort(lst):
	for i in range(len(lst)):
		swapped = False
		for j in range(len(lst) - 1):
			if lst[j] > lst[j+1]:
				lst[j],lst[j+1] = lst[j+1],lst[j]
				swapped = True
		if not swapped:
			break
	return lst
  • Big-O Complexity
    • Worst Case Time Complexity: O(N2)O(N^2)O(N2)﻿
      • Two nested for loops
    • Best Case Time Complexity: O(N)O(N)O(N)﻿ or O(N2)O(N^2)O(N2)﻿
      • O(N)O(N)O(N)﻿ if terminated early when no swaps
      • O(N2)O(N^2)O(N2)﻿ if not terminated early
  • Invariance and Stability
    • Invariance: At the end of i iterations, right most i elements are sorted and in-place.
    • Stability: Stable

Selection Sort

  • Graphics for Functionality
  • Implementation in Python
    def selection_sort(lst):
	for i in range(len(lst) - 1):
		for j in range(i+1, len(lst)):
			if lst[i] > lst[j]:
				lst[i],lst[j] = lst[j],lst[i]
	return lst
  • Big-O Complexity
    • Worst Case Time Complexity: O(N2)O(N^2)O(N2)﻿
      • Two nested for loops
    • Best Case Time Complexity: O(N2)O(N^2)O(N2)﻿
  • Invariance and Stability
    • Invariance: At the end of i iterations, left most i elements are sorted.
    • Stability: Unstable (Can be made stable by shifting elements and inserting)

Insertion Sort

  • Graphics for Functionality
  • Implementation in Python
    def insertion_sort(lst):
	for i in range(1,len(lst)):
		for k in range(i, 0, -1):
			if lst[k-1] > lst[k]:
				lst[k],lst[k-1] = lst[k-1],lst[k]
			else:
				break
	return lst
  • Big-O Complexity
    • Worst Case Time Complexity: O(N2)O(N^2)O(N2)﻿
      • One for loop and one inversion loop
    • Worst Case Time Complexity: O(N)O(N)O(N)﻿
      • If inversion loop is O(N)O(N)O(N)﻿ (when the list is already sorted), overall complexity is O(N+f(n))O(N+f(n))O(N+f(n))﻿ where f(n) is the inversion loop.
  • Invariance and Stability
    • Invariance: loop invariant condition is that the subarray A[0 to i-1] is always sorted.
    • Stability: Stable
